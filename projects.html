PROJECTS

Project 1: Create and submit a template for the CSS Zen Garden
Due: Thursday, February 19th

Concept and design a CSS design for the CSS Zen Garden website using techniques learned in class.

  1. Review the CSS Zen Garden website at http://www.csszengarden.com/
  2. Examine posted designs for inspiration.
  3. Download the XHTML and CSS templates from the CSS Zen Garden website .
  4. Build a complete design using CSS techniques learned in class.
  5. Upload your complete XHTML and CSS template to the CSS Zen Garden website.
  6. Add a link to your DMA442 website to the completed page on the CSS Zen Garden.

Project Requirements:

  1. All CSS must be accessed through an external style sheet only. Inline styling will not be accepted.
  2. All content must be positioned using CSS. Tables, &nbsp, transparent .gifs or <p> used for positioning will not be accepted and will result in a grade deduction.
  3. Completed style sheets should change the look of CSS Zen garden significantly. A basic 4-color change with link styles is unacceptable. You are encouraged to use images; illustrations and textures to create a completely different look for the Zen Garden.
  4.  All finished designed should have an overall concept associated with it. Remember, all finished designs will be critiqued and should stand strong against your peer’s questions. Think about what your doing and why. Some questions to ask yourself- how would this particular design element fit into my overall concept? By adding this, does it make my concept stronger or weaker? Also, don’t forget the importance of readability and usability.
  5. Your final page must validate in XHTML transistional. No Excuses!
  6. I will be checking the finished project in Firefox 3 and IE 7.
  7. All finished templates should be submitted by Thursday, February 19 before class. I will also collect your sketchbooks on the due date. If you have any questions or run into any problems, please talk to me before the final due date of the project.
Remember to review the regulations posted on the CSS Zen Garden website.

Project 2: Create a Database Driven Website using PHP and MySQL
Due: Thursday, March 26th

Concept and design a dynamic website with database driven elements using PHP and MySQL. The website can be your personal website, a  redesign of a website, or the creation of a new website. The completed design must be submitted for peer review and critique.

Project Requirements:
  1. The website must be at MINIMUM 10 pages.
  2. The website must use an external CSS style sheet
  3. The website must contain a dynamic contact form
  4. You must use PHP includes for the header and footer elements of the website. You should consider what elements will repeat from page to page- if it does, it must be included in the header or footer. A few example of what could be included in the header or footer are:
    Header-
                Ad banner
                Logo
                Callouts/news
                Navigation
                Breadcrumbs
                Last updated
               
    Footer-
                Contact info
                Copyright
                Designed by info
  5. You must design a logo for yourself or the organization chosen to place in the header of the website.
  6. All functionality needs to work. If anything is broken (ie- the submit button doesn’t work) the you will received a significant deduction in the final project grade. There’s no excuse! Test and retest your work.
  7. An element of or the completed site must be submitted for peer review. (EX: submit logo to logopond for review..) I will pass out a list in class of peer reviewed sites to submit your work to.
  8. All finished sites should be submitted by Thursday, February 19 before class. I will also collect your sketchbooks on the due date. If you have any questions or run into any problems, please talk to me before the final due date of the project.
